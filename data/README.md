# Data Notes

This repository is an anonymous review artifact and does not redistribute any raw benchmark data files.

To run the pipeline end-to-end, obtain the benchmark dataset separately and provide its path via the runner flags,
for example `--cases-path <path-to-cases.jsonl>`.

The included `results/` directory contains derived outputs (intermediate representations, generated notes, and
evaluation logs/metrics) used to reproduce the paper's tables and figures.
